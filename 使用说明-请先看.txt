# 77qicode 安装包使用说明 v3

版本: 3.1.0
更新日期: 2026-04-11

========================================
  重要提示
========================================

如果双击 77qicode.bat 后闪退，请使用 "诊断启动.bat"
它会显示详细的检查步骤，帮助定位问题。

========================================
  快速开始
========================================

1. 解压到任意位置

2. 双击 "诊断启动.bat" (推荐)
   或
   双击 "77qicode.bat"

3. 首次运行配置 API 密钥

========================================
  启动脚本说明
========================================

77qicode.bat
- 正常启动脚本
- 如果出错会显示错误信息并暂停

诊断启动.bat (推荐用于首次测试)
- 显示详细的检查步骤
- 检查 Node.js 是否可用
- 检查程序文件是否完整
- 显示详细的错误信息
- 不会闪退

========================================
  系统要求
========================================

必需: Node.js >= 18.0.0

验证 Node.js:
1. 打开命令提示符
2. 输入: node --version
3. 应该显示版本号（如 v20.11.0）

========================================
  故障排查
========================================

问题: 双击后闪退

解决方法:
1. 使用 "诊断启动.bat" 查看详细信息
2. 将显示的错误信息截图反馈

问题: 提示找不到 Node.js

解决方法:
1. 确认 Node.js 已安装
2. 在命令提示符输入: node --version
3. 如果提示错误，重新安装 Node.js

问题: 提示找不到文件

解决方法:
1. 重新解压安装包
2. 确保以下文件/文件夹存在:
   - dist/cli.js
   - node_modules/
   - .mcp.json

========================================
  手动启动方法
========================================

如果启动脚本无法使用，可以手动启动:

1. 在安装包目录打开命令提示符
2. 输入: node dist\cli.js
3. 查看错误信息
