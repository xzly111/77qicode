# 77qicode 安装包使用说明

版本: 3.1.0
类型: Windows 独立安装包

========================================
  快速开始
========================================

方式一: 使用启动器（推荐）
1. 双击 "启动器.bat"
2. 启动器会自动检测 Node.js
3. 如果未安装，会引导您安装
4. 安装完成后重新运行启动器

方式二: 使用安装程序
1. 双击 "installer.bat"
2. 按照提示完成安装
3. 安装后可以在任何位置输入 77qicode 启动

========================================
  系统要求
========================================

必需: Node.js >= 18.0.0

如果您的电脑没有 Node.js:
1. 访问 https://nodejs.org/
2. 下载 LTS 版本（推荐 v20.x）
3. 运行安装程序
4. 安装时勾选 "Add to PATH"
5. 安装完成后重启电脑

验证 Node.js 安装:
- 打开命令提示符
- 输入: node --version
- 应该显示版本号（如 v20.11.0）

========================================
  启动器功能
========================================

启动器会按以下顺序尝试找到 Node.js:

1. 系统 PATH 中的 Node.js
2. 常见安装路径:
   - C:\Program Files\nodejs\
   - C:\Program Files (x86)\nodejs\
   - %LOCALAPPDATA%\Programs\nodejs\
3. 手动指定路径
4. 引导安装 Node.js

========================================
  安装程序功能
========================================

安装程序会:
1. 检测 Node.js
2. 将程序安装到 %USERPROFILE%\77qicode
3. 添加到系统 PATH
4. 创建桌面快捷方式
5. 可选择立即启动

========================================
  首次使用
========================================

首次运行会进入 API 配置向导:

1. 选择 LLM 提供商
   - DeepSeek（推荐）
   - OpenAI
   - Anthropic Claude
   - 阿里云 Qwen
   - 自定义

2. 输入 API 密钥

3. 开始使用

========================================
  故障排查
========================================

问题 1: 启动器提示未找到 Node.js

解决方法:
1. 确认 Node.js 已安装
2. 打开命令提示符，输入: node --version
3. 如果提示"不是内部或外部命令":
   - 重新安装 Node.js
   - 确保勾选 "Add to PATH"
   - 重启电脑

问题 2: 安装程序无法运行

解决方法:
1. 右键点击 installer.bat
2. 选择"以管理员身份运行"

问题 3: 双击 .bat 文件闪退

解决方法:
1. 使用"启动器.bat"而不是直接运行
2. 启动器会提供详细的错误信息
