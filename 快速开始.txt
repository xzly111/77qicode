# 77qicode 快速开始

========================================
  3 步开始使用
========================================

1. 解压到任意位置

2. 双击 "77qicode.bat"

3. 首次运行配置 API 密钥

========================================

注意: 需要先安装 Node.js >= 18.0.0

如果双击后没有反应:
1. 打开命令提示符
2. 输入: node --version
3. 如果提示错误，请安装 Node.js
4. 下载地址: https://nodejs.org/

========================================
