========================================
  部署到测试机的步骤
========================================

1. 复制整个 77qicodess 文件夹到测试机

2. 确保测试机已安装 Node.js >= 18.0.0
   - 从 https://nodejs.org/ 下载安装
   - 安装时勾选 "Add to PATH" 选项

3. 双击 start-diagnostic.bat 进行诊断
   - 会自动搜索 Node.js
   - 显示详细的检查信息
   - 测试程序启动

4. 如果诊断通过,以后可以使用 start.bat 快速启动

========================================
  故障排查
========================================

问题: start-diagnostic.bat 提示找不到 Node.js

解决方案:
1. 检查 Node.js 是否已安装
   命令: node --version

2. 如果未安装,从 https://nodejs.org/ 下载安装
   重要: 安装时勾选 "Add to PATH"

3. 如果已安装但脚本找不到:
   - 查看 Node.js 安装位置: where node
   - 手动添加到系统 PATH 环境变量
   - 或重新安装 Node.js 并勾选 "Add to PATH"

4. 安装/配置后重启电脑

详细说明请查看: Node.js路径问题解决方案.txt
